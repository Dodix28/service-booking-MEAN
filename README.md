# Online Service Booking and Salon Management
*Your all-in-one salon booking and management solution.*

---

# Description
This project is a modern online service booking and salon management application designed for salons, hairdressers, and beauty professionals. It enables clients to schedule appointments quickly and conveniently, while providing salon staff with a centralized platform to manage services, bookings, and client information efficiently. By streamlining administrative tasks, reducing scheduling conflicts, and improving communication, this application helps salons save time, enhance customer experience, and operate more effectively.

---

# Technologies
| Frontend | Angular, TypeScript, CSS/HTML |
| Backend | NodeJS, Express |
| Database | MongoDB |
| Other | JSON import/export, REST API |

---

# Features
User-friendly Appointment Booking – Clients can quickly book appointments online.
Service Management – Salon staff can add, edit, or remove services.
Schedule Management – Easy calendar view for managing appointments and staff availability.
Client Management – Store client information, history, and preferences in one place.
Notifications & Reminders – Send appointment confirmations and reminders to clients.
Responsive Design – Works on desktop, tablet, and mobile devices.
Admin Dashboard – Overview of daily, weekly, and monthly bookings.
Search & Filter – Quickly find clients, appointments, or services.
Secure Login & Authentication – Protect staff and client data.
